# SLParser - StudentLibrary PDF Exporter 📚

Скрипт для браузерной консоли, позволяющий экспортировать полностью легально доступную пользователю книгу с сайта из режима "виртуального скроллинга" (ленивого рендеринга) в удобный, чистый и структурированный для печати PDF-документ.

## 📌 Что решает этот скрипт?
Многие образовательные платформы используют технологию *Virtual Scrolling* — текст рендерится только в видимой области экрана, а при прокрутке скрывается. Из-за этого:
- Невозможно выделить и скопировать несколько страниц текста подряд;
- Обычное сохранение страницы через `Ctrl + P` выдает пустые листы или обрывки текста;
- Неудобно читать с плохим интернетом (страницы подгружаются на лету).

## 🚀 Возможности
- **Извлечение полного текста:** Автоматически подгружает и склеивает все главы по API сайта на основе текущей сессии;
- **Титульный лист:** Генерирует обложку, авторов, год, издательство, ISBN и аннотацию для учебника;
- **Структуризация:** Восстанавливает иерархию заголовков и формирует кликабельное оглавление;
- **Очистка (Sanitizing):** Отключает шрифты и враждебные стили сайта (`display: none`), оформляя текст в классическом книжном стиле (с красной строкой, выравниванием по ширине и адаптивными картинками);
- **Сохранение в PDF:** ПодготавливаетDOM-дерево для идеального сохранения документа через стандартную функцию браузера "Распечатать" -> "Сохранить как PDF" (добавлены разрывы страниц `page-break-before`).

## 🛠 Инструкция по использованию

1. Зайдите на сайт электронной библиотеки в свой аккаунт.
2. Откройте нужную книгу (к которой у вас есть легальный доступ).
3. Нажмите `F12`, чтобы открыть панель разработчика (DevTools), и перейдите во вкладку **Console (Консоль)**.
4. Скопируйте весь код из файла `script.js` и вставьте его в Консоль (возле мигающего курсора `>`).
5. Нажмите `Enter`.
6. Дождитесь, пока желтый индикатор в правом верхнем углу пройдет по всем главам и покажет надпись **✅ Сборник успешно сформирован!**
7. Нажмите `Ctrl + P` (или Файл -> Печать) и выберите **"Сохранить как PDF"**. Убедитесь, что в дополнительных настройках печати стоят поля по умолчанию и включена печать фоновых изображений (для обложки).

## 👨‍💻 Особенности технической реализации
Скрипт использует встроенные в страницу глобальные переменные `reader_book` (ID книги) и `reader_access_token` (JWT токен сессии) для выполнения нативных `fetch` POST-запросов к `/api/3.0/mb4rdr-bff/retrieve/*`. Скрипт оснащен встроенной задержкой (`150ms`) между запросами, чтобы избежать блокировки (Rate Limit) со стороны сервера.
