# SLParser - StudentLibrary PDF Exporter 

## Parses a textbook from StudentLibrary for subsequent conversion to PDF

A browser console script that allows you to export legally accessible books from websites using "virtual scrolling" (lazy rendering) into a clean, well-structured PDF document ready for printing.

> **⚠️ Disclaimer**
> This script is created strictly for **educational purposes** and **personal use**.
> The script does not perform hacking, bypass DRM protection, or provide unauthorized access to any data. It works **only** with materials and books that the user already has legitimate access to (via purchased subscription or university access) within their authorized session.
> The author of this repository does not condone piracy and is not responsible for any misuse of this script. The entire codebase was generated by AI.

## 📌 What problem does this solve?
Many educational platforms use *Virtual Scrolling* technology — text is rendered only in the visible area of the screen and disappears when scrolled away. Because of this:
- You cannot select and copy multiple pages of text at once;
- Standard page saving via `Ctrl + P` results in blank pages or torn snippets of text;
- It's inconvenient to read with a poor internet connection (pages load on the fly).

## 🚀 Features
- **Full Text Extraction:** Automatically fetches and stitches together all chapters via the site's API based on your current session;
- **Title Page:** Generates a cover, authors, year, publisher, ISBN, and annotation for the textbook;
- **Structuring:** Restores heading hierarchy and generates a clickable Table of Contents;
- **DOM Sanitizing:** Strips away hostile CSS classes (`display: none`) and formats the text in a classic book layout (indented paragraphs, justified text, responsive images);
- **PDF Ready:** Prepares the DOM tree for perfect document saving using the standard browser "Print" -> "Save to PDF" function (adds `page-break-before` rules for chapters).

## 🛠 Usage Instructions

1. Log in to the digital library website with your account.
2. Open the book you want to read (that you have legal access to).
3. Press `F12` to open Developer Tools and go to the **Console** tab.
4. Copy the entire code from `script.js` and paste it into the Console (next to the `>` cursor).
5. Press `Enter`.
6. Wait for the yellow progress badge in the top right corner to iterate over all chapters and show **✅ Сборник успешно сформирован!** (Collection successfully generated!).
7. Press `Ctrl + P` (or File -> Print) and select **"Save to PDF"**. Ensure that in the advanced print settings, the margins are set to "Default" and background graphics are enabled (for the cover image).

## 👨‍💻 Technical Details
The script leverages the global variables `reader_book` (Book ID) and `reader_access_token` (JWT session token) already present on the page to perform native `fetch` POST requests to `/api/3.0/mb4rdr-bff/retrieve/*`. The script has a built-in delay (`150ms`) between requests to avoid triggering the server's Rate Limiter.
